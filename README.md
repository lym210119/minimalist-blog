# 极简主义博客

一个基于 Astro + Tailwind CSS 的极简主义博客，完美支持暗黑模式。

## 特点

- 🎨 **极简设计** - Less is More
- 🌙 **完美暗黑模式** - 纯黑背景
- ⚡ **极速加载** - Astro 静态生成
- 📱 **响应式** - 移动优先
- ♿ **无障碍** - 语义化 HTML

## 技术栈

- Astro 5.x
- Tailwind CSS 4.x
- Markdown

## 快速开始

```bash
# 安装依赖
pnpm install

# 开发模式
pnpm dev

# 构建
pnpm build

# 预览
pnpm preview
```

访问 http://localhost:4321

## 添加文章

在 `src/pages/posts/` 目录创建 `.md` 文件：

```markdown
---
title: "文章标题"
description: "简短描述"
pubDate: "2024-01-01"
---

# 文章内容

开始写作...
```

## 设计哲学

**极简主义** = 专注内容，去除干扰

- 黑白配色
- 清晰排版
- 充足留白
- 无多余装饰
